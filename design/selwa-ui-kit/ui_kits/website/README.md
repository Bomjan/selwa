# Selwa Website UI Kit

A high-fidelity recreation of the Selwa e-commerce website — a Bhutanese artisan marketplace. Covers core screens: homepage, products listing, product detail, artisan directory, and auth.

## Screens
- **Home** — hero, trust bar, category grid, featured products, artisan spotlight
- **Products** — filter sidebar, product grid, sort/filter bar
- **Product Detail** — image gallery, add to cart, artisan chip, tabs
- **Artisans** — directory with filter, artisan cards
- **Auth** — sign in / sign up split panel

## Usage
Load `index.html` to browse the full click-thru prototype.

## Design tokens
All tokens come from `../../colors_and_type.css`. Bootstrap Icons used for iconography.
